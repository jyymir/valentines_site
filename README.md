### üíå The "Valentine Trap" (Entrance)

- **Dynamic "Yes" Growth:** The "Yes" button grows in size every time the "No" button is avoided, making the right choice inevitable.

## üì± Mobile Responsiveness

The site is fully optimized for **iPhone, iPad, and Desktop**.

- **Touch Events:** Uses `touchstart` logic to ensure interactive buttons work on mobile devices.
- **Viewport Scaling:** Utilizes `viewBox` for SVGs and `max-width` containers to prevent horizontal scrolling.
- **Hamburger Menu:** A custom-animated 3-line menu that morphs into an "X" on mobile.
